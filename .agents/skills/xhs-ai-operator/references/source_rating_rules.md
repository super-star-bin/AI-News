# 来源分级规则

本文件与 `sources/source_rating_rules.md` 保持同一口径。

- S 级：官方博客、论文、GitHub、产品文档、Release Notes。可作为事实依据。
- A 级：研究者/创始人/核心员工公开发言、会议演讲。可作为观点和趋势依据。
- B 级：Reuters、FT、TechCrunch、The Verge、MIT Technology Review 等主流媒体。适合补充行业背景。
- C 级：HN、Reddit、Product Hunt、Discord 社区。只能作为线索和用户反馈。
- D 级：二手中文搬运、短视频爆料、未标来源截图。不能单独作为事实依据。

发布前尽量让核心结论至少有一个 S/A/B 级来源支撑。
