# 人工发布前审核清单

## 来源与事实

- [ ] 是否至少有一个一手来源。
- [ ] 是否标注不确定内容。
- [ ] 是否把社区讨论只作为线索。
- [ ] 是否区分官网信息和实际测试。
- [ ] 是否存在未实测却写成已实测。

## 表达风险

- [ ] 是否有“炸裂、封神、全网最强”等夸张词。
- [ ] 是否承诺收益或效果。
- [ ] 是否暗示所有人都适用。
- [ ] 是否省略重要限制条件。

## 平台合规

- [ ] 是否可能违规引流。
- [ ] 是否包含敏感联系方式。
- [ ] 是否泄露隐私或账号信息。
- [ ] 是否引用受版权保护全文。

## 发布准备

- [ ] 标题是否清楚。
- [ ] 封面是否对应正文。
- [ ] 7 页卡片是否有完整逻辑。
- [ ] 评论区引导是否合规。
